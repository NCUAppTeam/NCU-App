## Getting Start

First, you'll need [Node.js 12](https://nodejs.org/dist/latest-v12.x/) installed.

```
# If you haven't install expo-cli yet.
npm install -g expo-cli

# Install the dependency packages
npm install 

# Start the development server.
npm start
```
